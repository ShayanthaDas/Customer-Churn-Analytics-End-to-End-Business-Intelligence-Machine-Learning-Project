# Customer Churn Analytics: End-to-End Business Intelligence & Machine Learning Project

## Project Overview

Customer churn is one of the most critical challenges faced by subscription-based businesses. Losing existing customers often costs more than acquiring new ones, making customer retention a key business priority.

This project develops a complete Customer Churn Analytics solution using Excel, SQL, PostgreSQL, Python, Power BI, Tableau, and Machine Learning. The objective is to identify churn patterns, understand customer behavior, and build a predictive model that helps businesses proactively retain customers.

## Business Problem

The company has experienced a significant number of customer cancellations. Management wants to understand:

* Which customers are most likely to churn?
* What factors contribute to customer attrition?
* Which customer segments generate the most value?
* How can churn be reduced through data-driven decision making?

## Objectives

* Analyze customer behavior and churn trends.
* Identify key churn drivers.
* Segment customers based on value and risk.
* Develop predictive churn models using XGBoost.
* Create executive dashboards for decision-makers.
* Generate actionable business recommendations.

## Key Business Insights

* Month-to-month contracts exhibited the highest churn rates.
* Customers with shorter tenure were more likely to leave.
* High monthly charges increased churn probability.
* Long-term contracts significantly reduced churn.
* Customers with technical support subscriptions showed higher retention.

## Machine Learning Results

Model: XGBoost Classifier

Performance Metrics:

* Accuracy
* Precision
* Recall
* F1 Score
* ROC-AUC Score

The model successfully identified high-risk customers and can be used to support retention campaigns.

## Business Recommendations

1. Promote annual and multi-year contracts.
2. Prioritize retention campaigns for new customers.
3. Offer incentives to high-value customers.
4. Improve support services for at-risk segments.
5. Implement proactive churn monitoring using predictive analytics.
